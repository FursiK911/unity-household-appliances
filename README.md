# 🔥 VR Household Appliances

This project is a **demo assignment for Me Group** and is currently **still in development**.

It showcases a VR simulation of interacting with a gas stove, including match usage, burner control, ignition, and gas detection.

---

## 🎮 Features

* Pick up matches from a matchbox
* Turn burners on/off (including faulty ones)
* Ignite burners using a match
* Detect gas leaks with a gas analyzer

## 🛠️ Tech Stack

* Unity
* XR Interaction Toolkit
* C#

---

## 🚀 Getting Started

1. Clone the repository
2. Open the project in Unity
3. Load the main scene
4. Run in Play Mode or on a VR device

---

## 📌 Notes

* The project includes both working and faulty burners for simulation purposes
* Designed with VR interaction in mind (grab, trigger, proximity detection)
* Work in progress — features and behavior may change
